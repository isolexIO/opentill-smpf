# openTILL SMPF

### Structured Merchant Participation Framework • Powered by $DUC

> **The Merchant-Owned Financial Operating System**

---

## 🚀 Overview

openTILL is a modern, browser-based Point of Sale (POS) platform built to unify traditional payment processing with Web3 participation.

At its core is the **Structured Merchant Participation Framework (SMPF)** — a system that transforms everyday transaction activity into measurable network participation.

Merchants don’t just process payments.
They participate in — and benefit from — the system they power.

---

## ⚙️ Core Components

### 🖥️ openTILL POS

A full-featured, web-based POS system supporting:

* Credit & Debit Cards
* Cash Transactions
* EBT / SNAP
* Solana-based payments

Runs on:

* Tablets
* Laptops
* Desktop systems
* Compatible POS hardware (e.g., SkyTab)

---

### 🧠 SMPF (Structured Merchant Participation Framework)

A participation engine that:

* Tracks merchant activity
* Aligns usage with rewards
* Distributes $DUC based on contribution

---

### 💰 Digital Utility Credit ($DUC)

The native utility asset of the ecosystem used for:

* Feature access
* Staking for advanced capabilities
* Purchasing CHIP modules
* Governance participation

---

### 🧩 CHIP System (NFT-Based Modules)

Modular upgrades for the POS system:

Examples:

* Online Ordering
* Kitchen Display Systems (KDS)
* Customer Display
* AI Website Generator
* Third-party integrations (Uber Eats, DoorDash, etc.)

Merchants activate only what they need.

---

## 🔄 How It Works

1. Merchants process payments through openTILL
2. Transaction activity contributes to the network
3. SMPF evaluates participation
4. $DUC is distributed as incentives
5. Merchants use $DUC to:

   * Unlock features
   * Stake for higher tiers
   * Expand capabilities
6. Increased capability → more volume → network growth

> **Real Revenue In → Structured Value Out**

---

## 🧮 Reward Model

```
DUC_reward = (Transaction Volume × Participation Level × Network Factor) 
             / Total Active Merchants
```

* Rewards scale with real-world activity
* No artificial emissions
* Designed for sustainability
